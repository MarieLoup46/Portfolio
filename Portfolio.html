01    Compétences

Connaissances des composantes d'un ordinateur personnel

  • Processeur : Cerveau de l'ordinateur. Il gère les échanges de données entre les composantes.
  • Carte mère : Coeur de l'ordinateur. Elle supporte la plupart des composants et des connecteurs nécessaires au fonctionnement d'un PC.
  • Mémoire vive (RAM) : Assure un accès rapide et temporaire aux données de l'ordinateur.
  • Disque dur (ROM) : Utilisé pour stocker du contenu et des données numériques.
  • Boite d'alimentation : Fournit l'énergie nécessaire pour alimenter la carte mère et les divers éléments inclus dans l'unité centrale et doit permettre l'ajout ultérieur d'éléments supplémentaires.
  • Carte graphique : Carte d'extension dont le rôle est de produire une image affichable sur un écran.
  • Barettes : Support mémoire permettant de connecter de la mémoire vive à la carte mère.
  • Socket : Support pour le processeur qui sera connecté à la carte mère.
  • Ventilateur : Utilisé pour refroidir activement le boîtier. Il permet un flux d'air frais vers l'intérieur et d'air chaud vers l'extérieur ou améliore la circulation d'air sur le dissipateur thermique d'un composant particulier (ex.: le processeur).
  • Boitier : Contient les composants internes et les protège vis vis de l'extérieur.
  • Chipsets : Composants de la carte mère chargés de gérer les flux entre les différents composants de l’ordinateur.
    • Le northbridge (“pont nord”) : Gère les échanges de données entre les composants ayant besoin d’échanger un très grand nombre d’informations dans un laps de temps très faible (Processeur, Mémoire vive, Carte graphique.
    • Le southbridge (“pont sud”) : Gère les échanges de données entre les composants ayant besoin d’échanger un nombre plus faible d’informations : d’entrée et sortie.
  • Ports
    • Interne (PCI) : Permet de connecter des cartes d'extension sur la carte mère.
      • Les cartes graphiques
      • Les cartes son
      • Les cartes réseau
    • Externe : Permet de communiquer avec différents périphériques
      • USB
      • PS2
      • Parallèle
      • Série (ports RS232)
      • SATA
      • Réseau (Ethernet)
      • VGA (Signal analogique)
      • DVI
      • HDMI (Multimédia)
  • Périphériques d'entrée : Introduit des données ou des instructions : dans un ordinateur.
    • Souris et clavier
    • Écran tactile
    • Scanner
    • Boîtier de commande et manette de jeu
    • Appareil photo et caméra vidéo numérique
  • Périphériques de sortie : Qui se connecte à un ordinateur et génère une sortie, telle que du son, une image...
    • Imprimantes et télécopieurs
    • Moniteurs et projecteurs
    • Hautparleurs et casques
  • Périphériques d'entrée et de sortie : Un grand nombre de périphériques sont à la fois des périphériques d'entrée et de sortie.
    • Modem
    • Lecteur DVD
    • Disque dur
    • Clé USB

Connaissances du web

  • Le web 1.0 : Statique, centré sur la distribution d'informations.
  • Le web 2.0 : Dynamique (participatif), favorise l'échange entre les utilisateurs et la création de réseaux sociaux.
  • Le web 3.0 : Sémantique (qui a un sens), les publicités sont personnalisées en fonction du contexte et des besoins de chaque utilisateur.
  • Le web 4.0 : Intelligent, la possibilité de travailler avec des outils uniquement en ligne.
  • Protocole HTTP : Est un protocole de communication client-serveur (fonctionne sur le principe "requête-réponse") développé pour le World Wide Web (essentiellement au format HTML).
  • Protocole HTTPS : Est la variante sécurisée (de HTTP) par le chiffrement et l'authentification.
  • Requête HTTP : C'est un ensemble de lignes envoyé au serveur par le navigateur.
    La ligne comprend trois éléments devant être séparés par un espace :
    • La méthode
    • L'url
    • La version du protocole utilisé par le client
  • Réponse HTTP : Correspond au contenu du message. En réponse à la requête du client, le serveur envoie un ensemble de lignes de code au navigateur.
  • Méthodes de requête : La méthode GET passe les réponses saisies via l'URL tandis que la méthode POST passe les paramètres dans le corps de la requête.
  • Serveur web (statique ou dynamique) : C'est un ordinateur, souvent conçue pour fonctionner 24h/24, qui stocke les fichiers qui composent un site web.
  • URL : Adresse d'un site ou d'une page internet.
  • Adresse IP : Numéro d'identification de chaque appareil (serveur inclus) connecté à un réseau utilisant le protocole Internet.

Connaissances des réseaux informatique

Le réseau informatique désigne les appareils informatiques interconnectés qui peuvent échanger des données et partager des ressources entre eux. L'élément central des réseaux c'est l'INFORMATION.

Catégories de réseaux

  • LAN (Local Area Network) : Réseaux locaux.
  • PAN (Personnal Area Network) : Réseau personnel.
  • MAN (Metropolitan Area Network) : Réseaux métropolitains.
  • WAN (Wide Area Network) : Réseaux étendus.

Équipements réseau

  • Le répéteur : Régénère les signaux.
  • Un Switch : Permet de relier plusieurs segments d'un réseau informatique entre eux.
  • Un routeur : Équipement dont la fonction principale consiste à orienter les données à travers un réseau.
